Azure Fundamentals (AZ-900)

This repository documents the hands-on labs I complete while studying Azure Fundamentals and preparing for the Microsoft AZ-900 certification.

The goal of these labs is to build a solid understanding of core Azure services before progressing into Windows Server infrastructure projects and AZ-104 Azure Administrator studies.
