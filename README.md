# Azure Fundamentals Labs (AZ-900)

This repository documents my hands-on Azure labs while studying Azure Fundamentals and preparing for the Microsoft AZ-900 certification.

The purpose of these labs is to build a solid understanding of core Azure services and cloud concepts before progressing into Windows Server infrastructure projects and AZ-104 Azure Administrator studies.

---

